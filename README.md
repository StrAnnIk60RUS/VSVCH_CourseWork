# Онлайн-платформа для изучения иностранных языков – функции

- Фильтрация курсов по уровню сложности (A1–C2), языку, рейтингу.
- Сортировка курсов по дате добавления, популярности, оценке.
- Фильтрация студентов (для преподавателя) по курсу, успеваемости, дате последней активности.
- Сортировка студентов по имени, прогрессу, количеству выполненных упражнений.
- Напоминание о запланированных занятиях (push-уведомление или snackbar при входе) с возможностью настройки времени в личном кабинете.
- Построение графика прогресса студента (количество пройденных уроков/упражнений по дням/неделям).
- Формирование отчёта студента по пройденным урокам и успеваемости (PDF).
- Формирование отчёта преподавателя по успеваемости выбранной группы (PDF).
- Добавление курса в избранное (сохранение в localStorage).
- Просмотр детальной статистики по каждому упражнению (количество попыток, правильные/неправильные ответы).
- Сброс прогресса по курсу (для студента).
- Создание нового урока (для преподавателя) с добавлением текстового материала, видео, тестовых заданий.
- Автоматическая проверка упражнений с подсчётом баллов и обратной связью.
- Экспорт списка студентов (для преподавателя) в CSV.
- Сохранение настроек фильтров и сортировки (localStorage) с возможностью сброса.
- Интерактивный слайдер для выбора уровня сложности при поиске курсов.
- Вкладки (табы) для переключения между разделами статистики (графики, таблица результатов, рейтинг).
